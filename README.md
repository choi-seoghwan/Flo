# 기능 동작
## 주요기능
1. 스플래시 스크린
2. 음악재생화면 ( 음악정보, 가사(하이라이팅), SeekBar, play/stop )
3. 전체가사보기 (특정 가사 이동 토글 버튼, 닫기버튼, SeekBar, play/stop )
